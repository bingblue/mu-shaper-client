```cmd
$ npm start
```

## 版本列表（分支）

- [admin][3]  -  基于[vue-element-admin][4]的后台管理模板。
- [pc][5] - 基于[Vue][6]、[Element-UI][7]的前台展示模板。
- [uni][8] - 基于[uni-app][9]的小程序/H5模板。
- [admin-sass][10] - 和`admin`分支一样，不同的是使用Sass预编译器（不在维护更新）。
